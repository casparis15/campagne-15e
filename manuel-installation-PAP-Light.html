PAP Light – Installation

Application de porte-à-porte pour le 15e

Version actuelle : PAP Light – v1.01 (24/01/2025)

Accéder à l'application

Adresse : https://casparis15.github.io/campagne-15e/campagne-porte-a-porte.html?v=3.2

Ouvrez cette adresse dans le navigateur de votre téléphone, puis suivez les étapes ci-dessous pour installer l'application sur votre écran d'accueil.

Installation sur iPhone (iOS)

iPhone / iOS
  1. Appuyez sur le bouton « Partager » (carré avec une flèche vers le haut, en haut ou en bas de l'écran).
  2. Faites défiler les options (...), puis appuyez sur « + Sur l'écran d'accueil ».

L'icône de l'application apparaît alors sur votre écran d'accueil, avec les couleurs jaune–rose–vert du 15e.

Installation sur Android

Android

Option A – Notification automatique

  • Une notification « Ajouter à l'écran d'accueil » apparaît en bas de l'écran.
  • Appuyez sur « Installer » ou « Ajouter ».

Option B – Menu manuel

  1. Appuyez sur le menu (les 3 points verticaux) en haut à droite.
  2. Sélectionnez « Ajouter à l'écran d'accueil ».
  3. Confirmez en appuyant sur « Ajouter ».

Fonctionnement hors ligne

Bonne nouvelle : Une fois installée sur l'écran d'accueil, l'application fonctionne hors ligne, sans connexion internet. Vous pouvez donc l'utiliser sur le terrain même sans réseau.

Vérifier la version

En haut de l'écran de l'application, vous devez voir :

« PAP Light - v1.01 (24/01/2025) »

Si vous voyez une version plus ancienne :

  • Videz le cache de votre navigateur, puis rechargez l'application.

Données et sauvegarde

Toutes vos données sont :

  • Enregistrées uniquement sur votre téléphone
  • Conservées même si vous fermez l'application
Attention :
  • Si vous désinstallez l'application, toutes les données seront effacées.
  • Si vous videz le cache de votre navigateur, toutes les données seront également effacées.
